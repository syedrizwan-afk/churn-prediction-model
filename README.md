# Customer Churn Prediction

A Jupyter Notebook project for predicting customer churn using Logistic Regression, Random Forest, and XGBoost.

## Contents
- `churn_project_notebook.ipynb` — main notebook
- `requirements.txt` — dependency list
- `data/` — dataset folder
- `models/` — saved models

## How to Run
1. Open the notebook in Jupyter Notebook or VS Code.
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Run cells from top to bottom. It will auto-generate a synthetic dataset if none exists.
